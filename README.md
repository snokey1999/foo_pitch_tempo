# foo_pitch_tempo

一个基于 RubberBand 引擎的高质量变速变调与 loop 循环组件

## 功能说明
- **无损变速变调**：基于开源高质量 RubberBand 库，支持独立调节音频的播放速度和音高，极大地改善了音频质量（媲美 Reaper 等专业宿主），并支持多种算法模式。
- **调性检测 (Key Detection)**：基于 Essentia 库实现了快捷键和主菜单调性检测，支持置信度显示和用户微调。
- **A-B 段落循环**：可使用快捷键框选音频段落（设置 A 点和 B 点）进行无缝循环播放。
- **空间音频 (OpenAL 3D DSP)**：新增基于 OpenAL 的 3D 空间音频支持，支持 HRTF 与 EFX 预制混响效果。
- **快捷键绑定**：支持为所有功能（设置循环点、清除循环、增加/减少/重置音高和速度、调性检测等）绑定自定义快捷键。
- **跨版本兼容**：同时提供 32 位和 64 位支持，全面兼容 foobar2000 v1.6.x 以及全新的 v2.0+ 世代。
